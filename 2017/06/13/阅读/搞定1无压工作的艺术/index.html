第2章 掌控生活：横向管理工作流程的5个步骤

横向管理的5个步骤

  1. 收集：把一切会引起大脑注意的事务，不分轻重缓急（潜在的重要信息／任务／想法／行动／安排）一律从大脑中清空，作为原始材料放置工作篮中。
  2. 理清：它是什么？检查工作篮的材料，理清它们的意义，扔掉垃圾事件
  3. 整理：理清流程图和整理流程图一样，2分钟原则
  4. 回顾：思考回顾体现了GTD的最核心
